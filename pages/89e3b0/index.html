# 映射类型

PartialType()(部分声明): 函数返回一个类型（类），其中包含输入类型的所有属性，并都将其设置为可选。

当构建输入验证类型（也叫做数据传输对象或DTOs）时，在同一类型上构建create和update变体通常会很有用。

PickType()(拾取): 拾取部分类型

OmitType()(省略): 省略部分类型

IntersectionType(): IntersectionType()函数将两个类型组合为一个类型（类）

# @Module

@module() 装饰器接受一个描述模块属性的对象：

imports ：导入其他模块中导出的Providers，以实现共享

providers ：模块中所有用到的功能类，模块内共享实用；

controllers：控制器

exports：导出其他模块需要共享的Providers

# REPL

REPL 是一个简单的交互式环境，它接受单个用户输入、执行它们并将结果返回给用户。REPL 功能让你可以直接从你的终端检查你的依赖图并调用你的提供器（和控制器）上的方法。
